# 0.2.9 修正详情（2026-05-12）

## 触发背景

用户在做 ACM 题时，模型遇到超出能力范围的题目会反复尝试无效方法，浪费 token 和时间。需要一种机制让模型在确定自己大概率做不出来时坦诚告知用户，并提供替代帮助方式（分析题解）。

## 修改内容

### skills/acm/SKILL.md

1. **新增 `## Admitting Difficulty` 章节**：模型在合理尝试后确定无法解出时，坦诚告知用户（允许自嘲玩笑），询问是否有题解可帮忙分析。用户说"接着想"/"继续苦思冥想"则跳过此节，继续尝试不再询问。
2. **Quick Reference 表**：新增"接着想"/"继续苦思冥想"条目。
