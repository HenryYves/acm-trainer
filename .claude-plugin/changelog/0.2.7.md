## 设计决策

- **只记 C++ exe**：Python 可直接 `python file.py`，不需要配置。`solution_language` 为 `py` 时 setup 跳过此步。
- **exe_paths 镜像 code_paths**：同样的关键词→路径映射模式，保持一致性。
- **自动寻找**：从代码文件目录出发 Glob 搜索常见 exe 位置（x64/Debug、Debug、build 等），0 结果问用户，1 结果自动采纳，多结果让用户选。
